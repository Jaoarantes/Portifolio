Spotify

Este projeto é um clone da interface do Spotify, desenvolvido como forma de praticar e aprimorar minhas habilidades em front-end, especialmente com o framework Bootstrap. Utilizei HTML para a estrutura, CSS para estilização adicional e Bootstrap para garantir um layout responsivo e consistente em diferentes dispositivos.

Como foi meu primeiro projeto com Bootstrap, pude aprender sobre grids, componentes prontos (como cards e navbar) e a importância de um design mobile-first. O resultado final é uma réplica funcional e visualmente próxima ao Spotify, demonstrando como frameworks podem agilizar o desenvolvimento front-end.
